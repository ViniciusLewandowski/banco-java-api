Sistema Bancário - Spring Boot
API REST desenvolvida com Spring Boot simulando operações bancárias básicas.
Funcionalidades:
Cadastro de Cliente
Criação de Conta Corrente
Criação de Conta Poupança
Depósito
Saque
Transferência entre contas
Persistência em banco MySQL
Documentação via Swagger

Tecnologias:
Java 17
Spring Boot
Spring Data JPA
Hibernate
MySQL
Lombok
Swagger (OpenAPI)

Como executar:
Clone o projeto
Configure o application.properties
Rode a aplicação
Acesse: http://localhost:8080/swagger-ui/index.html
